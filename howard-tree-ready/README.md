# Howard Tree Pros

Simple static website for Howard Tree Pros.

## Open on GitHub

https://github.com/msimpson215/howardtreepros

## View locally

Open `index.html` in a browser, or run a local server:

```bash
npx serve .
```

## Update phone number

Search for `3145550100` and `(314) 555-0100` in `index.html` and replace with your real number.

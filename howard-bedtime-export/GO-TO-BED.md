# 60 seconds — then redeploy Render

## 1. Add index.html

https://github.com/msimpson215/howardtreepros → **Add file** → **Create new file**

Name: `index.html`

Paste everything from this folder's `index.html` → **Commit**.

## 2. Add these two tiny files (same way)

- `package.json` (paste from folder)
- `server.js` (paste from folder)

## 3. Render

Howard service → **Manual Deploy**

Build: `npm install` | Start: `npm start`

Open: https://howardtreepros.onrender.com

Your picture is already in the repo — the page uses it full-screen.
